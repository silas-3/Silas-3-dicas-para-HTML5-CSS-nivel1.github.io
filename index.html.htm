link para a terceira página, aode tem mais expecificação de como ela funciona. OBS.: Este link ira abrir uma nova guia, por que esta utilizando uma tag: [ target="_blank" ]

dentro de uma tag <a> da para adicionarmos dentro da função href= <a href="nome do index" rel="next" target="_self"></a>

Aonde a função [ rel="next" ] diz para o navegador que o link que digitamos é o próximo que tem que ser exibido.

A função [ rel="prev ou previus" ] quer dizer para o navegador que é o link para a página de pré-visualisação, e dando sentido [semantica] para o link.

A função [ rel="external" ] diz para o navegador que, é um link externo.

E a função [ target="_self"] diz para o navegador abrir na mesma página, dando sentido [semantica] para o link, no caso continua usando o site e abre uma nova página, porém detro do nosso site; usado para abrir outra página do nosso site.

E a função [ target="_blank" ] diz para o navegador abrir uma outra guia, a nossa página não é fechada, porém abre uma nova guia.
